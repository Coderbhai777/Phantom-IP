## 🛠️ Installation
Ensure you have Python 3 and a WSL Kali Linux environment installed.
```bash

git clone https://github.com/Coderbhai777/phantom-ip-suite.git
cd phantom-ip-suite

# Install dependencies
pip install requests beautifulsoup4 rich
```
